# config-default
Repository to save the initial settings of my projects.

## Directories

The directory structure must be as follows:
```
|-- config
    |-- gulpfile.babel.js
    |-- package.json
    |-- gulp
    |   |-- gulp-task-1.js
    |   |-- gulp-task-n.js
    |-- public
    |   |-- index.html
    |   |-- css
    |   |-- images
    |   |-- js
    |-- src
        |-- css
        |-- images
        |-- js
```
